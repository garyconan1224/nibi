## T1 文字结果页升级

**索引**：`docs/design/components/text_detail.jsx`（422 行）+ 现状 `TextResultPage.tsx`
**模型**：Sonnet
**分支**：`feat/t1-text-detail`
**子任务**：
- T1.1 结果契约升级：摘要 / 要点 / 金句输出结构化数组；金句必须可在原文中 substring 校验；要点必须带 `source_excerpt` + `char_start` / `char_end` / 段落序号，禁止近似跳转
- T1.2 「改写 · 翻译」tab 逐段对照：左原文段落，右当前改写/译文；按段落序号对齐，支持逐段复制，段数不一致时后端先对齐或标记缺失
- T1.3 原文定位交互：点击金句 / 要点 / 对照行时滚动到左侧原文精确位置并短暂高亮
- T1.4 多文对比 UI 补齐（IP.8.1 Compare Tab 已通后端）：复用 Taskboard Compare 的摘要 + 内容预览 + 字数表格，详情页弹窗不再只显示三列简表

**验收口径**：
- 用至少一条真实文字素材验证：点击任一金句能跳到原文中完全相同的文字范围。
- 用至少一条归纳型要点验证：点击要点跳到其 `source_excerpt`，不是跳到生成文案或段落近似位置。
- 改写 / 翻译结果按段落逐行展示；切换风格或语言时左侧原文段落位置稳定。
