
## 模块 6：图片分支

### 6.1 可勾选的分析任务（5 项）

| 任务 | 默认勾 | 说明 |
|---|---|---|
| 内容识别描述 | ✓ | 视觉模型分析，6 维度 |
| OCR 文字提取 | ✓ | 中英混合 / 手写 / 竖排 |
| 画面提示词生成 | ✓ | MJ / SD / JSON |
| 内容联想总结 | ✗ | 4 维度推导 |
| 多图对比 | ✗（动态出现） | 任务内图片 ≥ 2 张时才显示，需用户主动勾 |

### 6.2 基本信息读取

- 分辨率 / 格式 / 文件大小
- **EXIF**（7 字段）：拍摄设备 / 镜头 / 快门 / ISO / 光圈 / 拍摄地点 / 拍摄日期
- **EXIF 不存在时**（截图、AI 生成图、剥离过的图）：EXIF 区域显示"无 EXIF 信息"，其他分析照常进行；不强行推断

### 6.3 四个并行任务（互不依赖，可并发）

#### 任务 1：内容识别描述

视觉模型输出 **6 维度**：
- 主体（subject）
- 场景（scene）
- 色调（tones）
- 构图（composition）
- 风格（style）
- 细节（detail）

#### 任务 2：OCR 文字提取

**引擎选型**：
- **默认 PaddleOCR**（中文场景效果优，开源）
- 用户可在「设置 → 分析默认偏好」切换为"调用视觉大模型做 OCR"（更准但贵）
- 不再使用 Tesseract（中文一般）

支持：中英混合 / 手写体 / 竖排

输出：清洗排版后的文本，可导出 `.txt`

#### 任务 3：画面提示词生成

- 复用模块 4.2 的视觉模型分析逻辑
- 输出 MJ / SD / JSON 三种格式
- 自动打 7 维度标签（沿用 Phase 3C）

#### 任务 4：内容联想总结

**4 个固定联想方向**（用户多选）：
- 用途推断（usage）
- 设计风格分析（intent）
- 竞品分析（insight）
- 情绪解读

输出 4 维度：用途 / 目标受众 / 设计意图 / 竞品洞察

> ⏳ **延后事项**（不要忘）：联想方向是否支持"用户自定义补充第 5 条"或"完全自由文本输入"——用户使用一段时间后再决定是否扩展。当前阶段先做 4 个固定选项。

### 6.4 多图对比

**触发条件**：
- 任务内图片素材 ≥ 2 张时，"多图对比" 勾选项**自动出现在 preflight 中**
- 默认不勾，需用户主动开启

**对比流程**：
1. 各图先单独完成上面的 4 个并行任务
2. 横向对比 3 维度：
   - 风格一致性（是不是同一系列）
   - 差异点
   - 共同特征
3. 用户选输出格式：表格 / 报告

**对比入口位置**：
- 任务中心 → 「素材」子标签 → 多选图片素材 → 顶部出现"对比"按钮 → 点击 → 弹出对比结果对话框 / 新页面
- 对比结果**不作为独立素材入库**（只在对比页查看 / 导出）

### 6.5 图片结果交互界面（沿用设计稿）

**布局**（grid: 1fr 380px）：

```
┌──────────────────────────┬──────────────────────┐
│                          │ 提示词面板            │
│                          │ - MJ / SD / JSON tab  │
│                          │ - 一键复制            │
│                          │ - 收藏此图            │
│                          ├──────────────────────┤
│   原图全尺寸             │ ▼ 内容识别描述        │
│   （左侧主区）           │   6 维度 KV 行展示    │
│                          ├──────────────────────┤
│                          │ ▼ OCR 提取文字        │
│                          ├──────────────────────┤
│                          │ ▼ 联想总结            │
│                          │   4 维度              │
│                          ├──────────────────────┤
│                          │ ▼ EXIF 信息           │
└──────────────────────────┴──────────────────────┘
（底部）多图缩略图条：同任务内其他图片，点击切换
```

**交互**：
- 右侧每个分区默认展开，可折叠
- 提示词格式 tab 一键切换
- 收藏此图 → 加入复刻清单
- 底部缩略图条切换同任务其他图片

### 6.6 异常处理

| 异常 | 处理 |
|---|---|
| 图片损坏 / 格式不支持 | 提示重新上传 |
| 视觉模型 API 失败 | 重试 3 次，超限标注失败 |
| OCR 引擎不可用 | 自动回退到"视觉模型 OCR"路径 |
| EXIF 不存在 | 区域显示"无 EXIF" |

### 6.7 导出节点（绿色节点）

- 原图文件
- OCR 文字（.txt）
- 画面提示词（.json）
- 联想总结（.md）
- 多图对比报告（.md / .json）
