
## 模块 4：视频分支

### 4.1 可勾选的分析任务（4 项）

| 任务 | 说明 | 默认勾 |
|---|---|---|
| 画面提示词生成 | 截帧 → 视觉模型 → MJ/SD/JSON 提示词 | ✓ |
| 视频文案总结 | 三条路径选一 | ✓ |
| 字幕导出 | .srt 文件 | ✓ |
| 音乐分析 | BPM / 调性 / Suno-Udio 提示词（联动模块 5） | ✗ |

### 4.2 画面提示词生成

#### 4.2.1 截帧模式（用户在 Preflight 抽屉选）

**模式 A：按秒截帧**
- 参数：每 N 秒一帧（默认 5）/ 最大帧数（默认 100）
- 用 ffmpeg 均匀截取
- **后置自动过滤相似帧**（用户无感知），算法：相邻帧像素差异 / phash 比较

**模式 B：AI 镜头分析（默认）**
- 用 **PySceneDetect**（开源场景检测库）自动识别镜头切换点
- 弹窗展示「检测到 N 个镜头」让用户确认
- 用户选每镜头取几帧：
  - **2 帧模式**：首帧 + 尾帧（适合简单运镜）
  - **3 帧模式**（默认）：首帧 + 中间帧 + 尾帧（适合复杂镜头）
- 不再过滤相似帧（按切换点截不会重复）

#### 4.2.2 视觉模型分析

每帧送入用户配置的视觉大模型，输出 5 维度：
- 主体 / 风格 / 光线 / 色调 / 构图

**异常处理**：API 失败自动重试 3 次，超限标注"该帧分析失败"并跳过。

#### 4.2.3 提示词输出（当前阶段：3 种格式）

| 格式 | 内容 |
|---|---|
| **Midjourney（MJ）** | 含 `--ar` / `--style` / `--s` 等 MJ 专属参数 |
| **Stable Diffusion（SD）** | 适配 SD 关键词风格 |
| **JSON** | 含 timestamp / title / prompt_mj / prompt_sd / tags 字段 |

**语言**：默认英文；Preflight 抽屉可切换中英双语。

#### 4.2.4 延后事项（不要忘记）

⏳ **视频运镜提示词** — 第 4 种格式，描述运动起点和终点（配合 AI 镜头分析的 2 帧/3 帧模式），当前阶段不做，**后续 AI 导演模块开发时一并补上**。

#### 4.2.5 自动打标签入库

提示词按 7 维度自动归类（沿用 Phase 3C 已实现的标签库）：
风格 / 光线 / 构图 / 色调 / 镜头 / 人物 / 场景

### 4.3 视频文案总结（3 条路径都做）

#### 4.3.1 字幕预处理（前置步骤）

```
检测视频是否自带字幕
  ├─ 有字幕（内嵌字幕轨 / 外挂 .srt .ass）
  │     → 直接提取字幕文本，原文使用，不清洗
  │     → 字幕文件可单独导出
  └─ 无字幕
        → 提取音频轨道 → 音频文件可单独导出
        → Whisper 转写（带时间戳）
        → 字幕清洗 → 清洗后字幕可单独导出
```

**字幕清洗规则**：
- 去语气词（嗯、啊、那个）
- 合并不必要的短句
- 修正明显的识别错误
- **专有名词修正**：把任务级背景信息中填写的"专有名词"作为参数注入清洗 LLM 的 prompt，让 LLM 在清洗时优先匹配（不修改 Whisper 本身的 initial_prompt 参数，因为后处理更可控）

#### 4.3.2 三条总结路径（用户在 Preflight 抽屉选）

**路径 1：字幕直接总结**
- 流程：字幕 → 超长文本分段 → 套视频类型模板 → LLM 总结
- 输出：标题 / 摘要 / 要点列表 / 金句
- 特点：最快、最便宜
- 适合：纯语音内容

**路径 2：音视频合并总结（默认，最详细）**
- 流程：字幕 + 截帧画面描述（复用 4.2 截帧）→ 按时间戳对齐合并 → 套模板 → LLM 总结
- 输出：摘要 / 要点 / **画面引用说明（带截图）**
- **音乐分析联动**：路径 2 下如果勾了音乐分析，音乐分析必跑

**路径 3：视频模型直接分析（最轻量）**
- 用户选视频大模型：Gemini 1.5 Pro / GPT-4o / Qwen-VL-Max 等
- 整段视频送入模型（无需截帧）
- 输出：视频简单描述（质量依赖模型）
- 界面标注"所用模型名称"

#### 4.3.3 视频类型 → 总结模板

视频类型从**任务级背景信息**的「内容类型」字段读取（不重复让用户选）：

| 内容类型 | 输出侧重 |
|---|---|
| 课程 | 知识点列表 / 重点概念 / 学习建议 |
| 宣传片 | 核心卖点 / 目标受众 / 视觉亮点 |
| Vlog | 地点 + 事件亮点 / 情绪氛围 |
| 访谈 | 受访者观点 / 金句摘录 / 话题梳理 |
| 新闻报道 | 事件背景 / 关键信息 / 时间线 |
| 其他 | 通用模板（摘要+要点+金句） |

### 4.4 视频结果交互界面（沿用设计稿）

**布局**（grid: 1fr 300px）：

```
┌─────────────────────────────────┬───────────────┐
│  顶部：导航 + 标题 + .srt/.md 导出│ 右侧浮动面板  │
├─────────────────────────────────┤   (300px)     │
│  7 维度标签面板（ItemTagsPanel） │  当前帧预览   │
├─────────────────────────────────┤   (含收藏星)  │
│                                 │  帧标题+副标  │
│  视频播放器（16:9，可点击播放） │  当前帧 tags  │
│                                 ├───────────────┤
├─────────────────────────────────┤  提示词格式   │
│  轨 1：镜头缩略图（点击跳转）   │  tab MJ/SD/JSON│
│  轨 2：字幕（自动高亮）         ├───────────────┤
│  轨 3：提示词区间（每帧一张卡） │  提示词文本   │
│                                 │  (mono 字体)  │
│                                 ├───────────────┤
│                                 │  一键复制 ★   │
│                                 │  收藏此帧     │
│                                 │  帧 N/M       │
└─────────────────────────────────┴───────────────┘
```

**核心交互**：
- 视频播放时，三轨自动同步滚动（缩略图、字幕、提示词区间都跟着当前帧高亮）
- 点击任意位置（缩略图/字幕行/提示词卡）→ 视频跳转到对应时间
- 右侧面板自动更新当前帧的提示词
- 提示词格式 tab 一键切换（MJ ↔ SD ↔ JSON）
- 「收藏此帧」加入复刻清单（关联到模块 8 的 AI 导演）

**后续可能改的方向**（先做现状，根据实际使用反馈再调整）：
- 第四轨"音乐情绪"
- 播放器倍速
- 提示词内联编辑

### 4.5 异常处理

| 异常 | 处理 |
|---|---|
| 视觉模型 API 失败 | 重试 3 次，超限标注"该帧失败"并跳过 |
| 字幕完全无人声（Whisper 转空） | 提示用户检查音频或切换为音乐分析模式 |
| 文本超长 | 自动分段处理，结果合并 |
| 视频时长超 X 分钟（视频模型路径） | 提示用户改用路径 1/2 或切片 |

### 4.6 导出节点（绿色节点）

- 原视频文件
- 提取的音频文件（.wav / .mp3）
- 字幕文件（.srt / .txt）—— 清洗前后版本都可
- 文案总结（.md）
- 画面提示词（.json，含全帧时间戳 + MJ/SD/tags）
