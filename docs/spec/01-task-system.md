
## 模块 1：任务系统（顶层架构）

### 1.1 核心概念

- 一个**任务** = 一个独立工作空间
  - 代码层叫 `workspace`，UI 文案统一叫"任务"
- 用户可以**同时拥有多个任务**，任务列表页平铺展示，支持搜索切换
- 一个任务里**素材类型混搭**：视频 + 音频 + 图片 + 文字 任意组合，数量不限
- **去掉历史 `project` 这一层**——`WorkspaceRecord.project_id` 字段清理掉，workspace 自治

### 1.2 任务状态

| 状态 | 含义 |
|---|---|
| `active` | 任务存在，可以添加/查看素材，**不一定全部分析完** |
| `processing` | 当前有素材正在跑分析任务 |
| `analyzed` | 任务内所有素材分析都完成（**注**：分析完成 ≠ 任务结束，后续还能做复刻、AI 对话、生成新内容） |
| `archived` | 用户主动归档 |
| `trashed` | 软删除（进入垃圾桶，主列表不显示） |

### 1.3 任务字段

- **基本信息**：任务名 / 创建时间 / 更新时间 / 状态
- **背景信息（5 字段，全部可选填）**：
  - 内容类型（课程 / 会议 / 宣传片 / Vlog / 访谈 / 纯音乐 / 其他）
  - 参与人物（自由文本，例："Hugo · 影视飓风"）
  - 主题背景（自由文本，例："Q2 数码产品开箱评测"）
  - 专有名词（用于提升 Whisper / LLM 识别准确率）
  - 分析目的（复刻参考 / 竞品分析 / 内容学习 等）
- **素材列表**（每个素材类型不同，详见模块 4-7）
- **收藏夹**（item_id 列表，用于后续复刻清单）
- **提示词版本栈**（每个素材的提示词迭代历史）
- **任务级 LLM 对话历史**（独立文件存储）

### 1.4 任务操作

| 操作 | 说明 |
|---|---|
| 新建任务 | 弹窗输入任务名，创建空任务 |
| 重命名 | 任务详情页可改名 |
| 编辑背景信息 | 模态弹窗逐字段编辑 |
| 添加素材 | 详见模块 2 |
| 软删除 | 进入垃圾桶，**不立即删素材文件** |
| 从垃圾桶恢复 | 恢复到原状态 |
| 彻底删除 | 在垃圾桶里清空或单个删——**物理删除任务记录 + 所有关联素材文件**（下载的视频、截帧、字幕、生成结果都删） |
| 导出复刻工作包 | 详见模块 8 |

**垃圾桶入口位置**：放在「设置」页里，不污染主列表。

### 1.5 任务级 LLM 对话

- **对话面板** 顶部有一排「上下文素材」多选 chip
- **默认行为**：用户必须选 ≥ 1 个素材作为上下文（避免漫无目的的提问）
- **快捷按钮**：「全任务上下文」一键勾选所有素材
- **token 兜底**：选完后内容仍超出模型上下文时，后台自动 RAG 检索相关片段，界面顶部标注「上下文已自动精简」
- **对话历史**：按任务独立存储（`data/chats/<workspace_id>.jsonl`，沿用现有 chat 路由设计）

**典型业务场景**（写进 spec 帮助后续设计师/开发理解）：

| 场景 | 选几个素材 | 典型问题 |
|---|---|---|
| 跨素材对比 | 2-3 个 | "这两个视频开头风格有什么不同？" |
| 单素材聚焦 | 1 个 + 时间范围 | "这个视频 0-10 秒的提示词怎么优化？" |
| 跨素材归纳 | 3+ 个同类 | "这 3 张海报色调共同点是什么？" |
| 全任务洞察 | 全选 | "我这个任务里所有素材最高频的标签是什么？" |

### 1.6 「AI 导演」（复刻功能）暂定策略

- **当前阶段不做**——产品需求还没想清楚，先记录方向
- **未来定位**：独立的一级模块，**不再放进 Taskboard 子标签**（避免功能重复）
- **预期包含**：收藏帧（复刻清单）/ 提示词版本 / A/B 对比 / 风格 DNA / AI 对谈
- **扩展方向**：将来可能接入可灵 / 即梦 等视频图片生成模型 API，实现一键复刻
- **过渡期做法**：当前阶段任务里"能收藏帧 + 能存提示词版本"就行，对比/风格报告/生成对接 都延后

### 1.7 一级导航（侧边栏）— 精简后 4 项

| 入口 | 说明 |
|---|---|
| 任务中心（Taskboard） | 核心入口，进入后看到当前任务的子标签页 |
| 资料库（跨任务知识库检索） | Phase 3B 已有，支持跨工作空间搜字幕/标签/提示词 |
| AI 导演 | **当前阶段隐藏或灰显**，等复刻功能开发再开启 |
| 设置 | 模型管理 / 全局偏好 / 垃圾桶 |

**搜索入口**走顶部 topbar，不占侧边栏位置。

**砍掉的入口**及理由：
- ~~工作台~~ → 合到任务中心首页
- ~~处理中~~ → 合到任务中心的「队列」子标签
- ~~结果~~ → 进具体素材详情页看
- ~~分镜~~ → 进视频详情页的三轨时间轴
- ~~12 屏概览~~ → 砍掉（用途不清晰）

### 1.9 素材卡片展示信息

- 缩略图（视频取首帧 / 图片取原图缩略 / 音频用波形占位 / 文字用文档图标）
- 类型徽章（视频/音频/图片/文字，配色：粉/紫/蓝/琥珀）
- 状态点（done 绿 / running 黑 / queued 灰 / error 红）
- 来源（B 站链接 / 本地文件名 等）
- 元信息（时长 / 大小 / 分辨率 等，按类型不同）
- 前 3 个 tag（超出显示 "+N"）
