# 笔记功能流程图与 UI 布局架构

```text
                                [ 链 接 ]
                                    │
                                    ▼
                 [ 自动分析 (音频，视频，图片，文字) ]
                                    │
                                    ▼
          [ 下载 (自动分析的结果，根据情况选择合适的下载方式) ]
                                    │
                                    ▼

╔═══════════════════════════════════════════════════════════════════╗
║                             内容处理                              ║
╚═══════════════════════════════════════════════════════════════════╝

   音频 ──────> 转码 ────────> MD

   文字 ──────> MD

            ┌─> 文字型 ──────> 内容转写 MD
   图片 ────┤
            └─> 非文字型 ────> 视觉模型去做描述

            ┌─> 分离音频 ────> 转码 ────────────> MD
   视频 ────┤
            └─> 画面截图 ────> 图片内容简单识别 ─> MD
                (每几帧截图设置页设置)

                                    │
                                    ▼

╔═══════════════════════════════════════════════════════════════════╗
║                             生成笔记                              ║
╚═══════════════════════════════════════════════════════════════════╝

音频：MD源内容 ──> 标准总结 ──> MD展示和富文本展示，同时还有音频播放器，
                                播放音频时下面会有字幕自动跟随


                      ┌───────────────────────────────────────┐ 
                      │  [音频播放]      [实时字幕]           │ 
                      │                                       │   1.源 md 文件点击跳转看
                      │ ┌──────────────────────────┐ ┌─────┐  │   2.选择其他总结方式
                      │ │                          │ │     │──┼─  3.ai 问答
                      │ │ 标准总结的MD与富文本切换 │ │     │  │   4.等等
                      │ │                          │ │     │  │
                      │ └──────────────────────────┘ └─────┘  │
                      │  md 可带时间码，                      │
                      │  点击跳转到对应音频时间               │
                      └───────────────────────────────────────┘

─────────────────────────────────────────────────────────────────────

文字：源 MD 内容 ──> 标准总结 ──> MD 内容和富文本展示


                      ┌───────────────────────────────────────┐ 
                      │ ┌──────────────────────────┐ ┌─────┐  │   1.源 md 文件点击跳转看
                      │ │                          │ │     │  │   2.选择其他总结方式
                      │ │ 标准总结的MD与富文本切换 │ │     │──┼─  3.ai 问答
                      │ │                          │ │     │  │   4.等等
                      │ └──────────────────────────┘ └─────┘  │ 
                      └───────────────────────────────────────┘
                            
                      1.md 内容都可编辑，有个编辑器，加粗等等各种功能。
                      2.富文本对应自动带格式

─────────────────────────────────────────────────────────────────────

图文：
          文字源 MD 内容 ──────────┐
                                   ├──> 标准总结 ──> MD 内容和富文本展示
      文字图片内容转写 源MD内容 ───┘

                      ┌───────────────────────────────────────┐ 
                      │ ┌──────────────────────────────────┐  │   1.源 md 文件点击跳转看
                      │ │ 图片预览，点击放大               │  │   2.选择其他总结方式
                      │ └──────────────────────────────────┘  │   3.ai 问答
                      │ ┌──────────────────────────┐ ┌─────┐  │   4.等等
                      │ │                          │ │     │──┼─  
                      │ │ 标准总结的MD与富文本切换 │ │     │  │ 
                      │ │                          │ │     │  │
                      │ └──────────────────────────┘ └─────┘  │
                      └───────────────────────────────────────┘
                                 带文字图片内容


          文字源 MD 内容 ──────────┐
                                   ├──> 标准总结 ──> MD 内容和富文本展示
非文字图片视觉模型去做描述 MD ─────┘

                      ┌───────────────────────────────────────┐ 
                      │ ┌──────────────────────────────────┐  │   1.源 md 文件点击跳转看
                      │ │                                  │──┼─  3.ai 问答
                      │ │ 图片富文本，带描述               │  │   4.等等
                      │ │                                  │  │
                      │ └──────────────────────────────────┘  │
                      └───────────────────────────────────────┘
                                 非文字图片内容
                      
                      1.md 内容都可编辑，有个编辑器，加粗等等各种功能。
                      2.富文本对应自动带格式
                      3.点击图片可以添加到 md 文件中

─────────────────────────────────────────────────────────────────────

视频：源 MD 内容 ────> 标准总结 ────┐
                                    │
      带描述截图 ───────────────────┴─> 带图标准总结 MD 内容和富文本展示
                                        文字中在合适的位置插图


                      ┌───────────────────────────────────────┐ 
                      │ ┌──────────┐ ┌────────────────┐ ┌───┐ │   1.源 md 文件点击跳转看
                      │ │ 视频播放 │ │ 标准总结的MD与 │ │   │ │   2.选择其他总结方式
                      │ ├──────────┤ │ 富文本切换     │ │   │─┼─  3.ai 问答
                      │ │ 实时字幕 │ │                │ │   │ │   4.等等
                      │ ├──────────┤ ├────────────────┤ │   │ │ 
                      │ │          │ │ md 可带时间码，│ │   │ │
                      │ └──────────┘ │ 点击跳转到对应 │ │   │ │
                      │              │ 时间画面       │ └───┘ │
                      │              └────────────────┘       │
                      └───────────────────────────────────────┘
                                    视频内容
                      
                      1.md 内容都可编辑，有个编辑器，加粗等等各种功能。
                      2.富文本对应自动带格式
                      3.视频播放可以截帧，导入到 md 文件中
                      4.点击字幕和对应总结端的内容可以跳转到对应时间轴的视频

                                    │
                                    ▼
╔═══════════════════════════════════════════════════════════════════╗
║                               导  出                              ║
╚═══════════════════════════════════════════════════════════════════╝
```
