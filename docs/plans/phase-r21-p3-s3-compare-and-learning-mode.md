## 目标（一句话）

让多份总结能并排对比；让学习视频的转录正文旁能由 AI 推荐 + 用户手动选时间轴帧插入截图。

## 关键设计点（不变项）

### 1. 总结对比模式

- 列表区每行加 checkbox，勾选 2-3 份后顶部出现 `[⇄ 进入对比]` 按钮
- 进入后右侧主显示区由单栏变为 N 栏（2 栏或 3 栏，flex 平分）
- 每栏顶部展示模板名 + 版本号 + 总结用背景（折叠）
- `[退出对比]` 回单栏
- 最多支持 3 栏（屏幕宽度限制）

### 2. 视频学习模式按需补图（用户决定 Q1 = c）

行为：
- 学习模式视频在「原始产出」tab 看到：转录正文（分段）+ 一个折叠的「关键帧时间轴」
- AI 在 pipeline 后预跑一遍**轻量画面变化检测**，给转录段落自动建议「这段适合插图」标记
- 用户在转录正文每段右侧有「📷 插图」按钮，点击弹**时间轴帧选择器**，可：
  - 用 AI 推荐的关键帧
  - 或拖时间轴自定义任意时刻截一帧
- 插入后该帧出现在转录段落下方，可删除可替换
- 插入的帧持久化到 item 的某个字段（具体表设计进入阶段时定）
