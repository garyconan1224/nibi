## 范围概述

v1.1 设计契约提到的"标签库 7 维度"功能。每个工作空间产物（视频/图片/音频/文本）通过 LLM 自动打 7 个维度的标签，用户可手动校正；标签作为筛选/聚合维度供前端使用。

完成 3C 后，**信息架构定型**——这是 Claude Design 介入做完整 UI 翻新的最佳时机。

## 7 维度（待最终确认）

来自 spec v2 §3 触发条件 + 总规划 §7，预估维度（实际开工前应与用户对齐）：
- 内容类型（教程/访谈/解说/纪实/Vlog/...）
- 主题领域（科技/人文/财经/...）
- 难度等级（入门/进阶/专家）
- 时长档位（短/中/长）
- 信息密度（高/中/低）
- 情绪基调（中性/激励/批判/...）
- 用户自定义标签（自由文本数组）
