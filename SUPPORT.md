# Support

Nibi is a local-first creator tool. Support is community best-effort.

## Good Issue Reports

Please include:

- Operating system and version.
- Python, Node, and pnpm versions.
- The command you ran.
- Full error output.
- Whether the issue requires a model API key, local media file, or third-party platform URL.

Do not upload API keys, cookies, private links, downloaded videos, local databases, or personal workspace content.

## Scope

The project can help with:

- Installation and local startup problems.
- Reproducible UI or API bugs.
- Graceful failure for unsupported platforms or missing dependencies.
- Documentation gaps.

The project cannot guarantee:

- Access to private or restricted third-party content.
- Compliance for a user's specific copyrighted material.
- Availability of unofficial third-party endpoints.
- Free model/API usage.
